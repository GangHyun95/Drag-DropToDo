# React Drag & Drop TodoList

## 🔎 프로젝트 소개
Drag & Drop 이 가능한 TodoList 입니다.

## 🔨 사용 기술

* `React`
* `React-beautiful-dnd`
* `JavaScript`
* `TypeScript`
* `Recoil`
* `Recoilpersist`
* `Styled-Components`
* `react-hook-form`

#### 보드간 Drag & Drop 가능

#### TodoList 작성 및 Drag & Drop 가능
